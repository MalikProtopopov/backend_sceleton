## Общие правила

### Форматы изображений
- **Формат**: JPG, PNG, WebP
- **Размер файла**: до 5 МБ
- **Рекомендуемые размеры**:
  - Обложки статей/кейсов: 1200×630 px (соотношение 1.91:1 для соцсетей)
  - Фото сотрудников: 400×400 px (квадрат)
  - Логотип: SVG или PNG с прозрачным фоном, 200×60 px
  - OG-изображения: 1200×630 px

### SEO-поля (присутствуют во всех сущностях с локализацией)

| Поле | Тип | Лимит | Описание |
|------|-----|-------|----------|
| `meta_title` | Строка | **70 символов** | Заголовок для поисковиков (тег `<title>`) |
| `meta_description` | Строка | **160 символов** | Описание для поисковиков |
| `meta_keywords` | Строка | 255 символов | Ключевые слова (через запятую) |
| `og_image` | URL | 500 символов | Картинка для соцсетей (Open Graph) |

### Языковые версии (Locales)
- Поддерживаемые локали: `ru`, `en`, `de`, `fr` и др.
- Каждая сущность имеет отдельные поля для каждого языка
- **Обязательно**: русская версия (`ru`)

## 1. Услуги (Services)

### Основные поля

| Поле | Тип | Обязательное | Лимит | Описание |
|------|-----|--------------|-------|----------|
| `icon` | Строка | Нет | 100 симв. | Название иконки (например, `briefcase`, `chart-line`) |
| `image_url` | URL | Нет | 500 симв. | Главное изображение услуги |
| `price_from` | Число | Нет | ≥ 0 | Цена "от" (в копейках/центах) |
| `price_currency` | Строка | Да | 3 симв. | Валюта: `RUB`, `USD` |
| `sort_order` | Число | Нет | — | Порядок сортировки (0 = первый) |
| `is_published` | Да/Нет | Да | — | Опубликовано на сайте |

### Локализованные поля (для каждого языка)

| Поле | Тип | Обязательное | Лимит | Описание |
|------|-----|--------------|-------|----------|
| `title` | Строка | **Да** | **255 символов** | Название услуги |
| `slug` | Строка | **Да** | 255 симв. | URL-адрес (латиница, дефисы): `web-development` |
| `short_description` | Строка | Нет | **500 символов** | Краткое описание для карточки |
| `description` | Текст | Нет | Без лимита | Полное описание (поддерживает HTML/Markdown) |
| + SEO-поля | — | — | — | См. выше |

### Теги услуги (ServiceTag)

| Поле | Тип | Лимит |
|------|-----|-------|
| `tag` | Строка | 100 символов |

**Пример тегов**: `React`, `TypeScript`, `AWS`, `PostgreSQL`

### Что собрать для каждой услуги

```
□ Название услуги (RU/EN)
□ URL-slug (латиница)
□ Краткое описание (до 500 символов)
□ Полное описание (без лимита, с форматированием)
□ Иконка (название из набора иконок)
□ Изображение услуги (1200×800 px)
□ Цена "от" (опционально)
□ Теги/технологии (список)
□ SEO: meta_title (до 70), meta_description (до 160)
□ OG-изображение (1200×630 px)
```

## 7. SEO-настройки страниц

### Поля SEO-маршрута

| Поле | Тип | Обязательное | Лимит | Описание |
|------|-----|--------------|-------|----------|
| `path` | Строка | **Да** | 500 симв. | URL страницы: `/about`, `/services/web` |
| `locale` | Строка | **Да** | 5 симв. | Язык: `ru`, `en` |

| `meta_title` | Строка | Нет | 70 симв. | SEO заголовок |
| `meta_description` | Строка | Нет | 160 симв. | SEO описание |
| `canonical_url` | URL | Нет | 500 симв. | Канонический URL |
| `robots_index` | Да/Нет | Да | — | Индексировать страницу |
| `robots_follow` | Да/Нет | Да | — | Следовать по ссылкам |
| `structured_data` | JSON | Нет | — | JSON-LD разметка |
| `sitemap_priority` | Число | Нет | 0.0-1.0 | Приоритет в sitemap |
| `sitemap_changefreq` | Выбор | Нет | — | Частота: `daily`, `weekly`, `monthly` |
| `include_in_sitemap` | Да/Нет | Да | — | Включить в sitemap |

### Типичные страницы для SEO-настройки

```
□ Главная (/)
```
